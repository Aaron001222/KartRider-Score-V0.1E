# 🏁 跑跑卡丁車計分系統
 
一個簡單好用的跑跑卡丁車比賽計分工具，用 Python 寫的桌面程式。

## 主要功能

- 支援 8 名玩家同時計分
- 11 種顏色可以選，每個玩家都有專屬顏色
- 即時顯示總分和每輪分數
- 按鈕式排名：左鍵點擊分配名次，右鍵清除
- 可以撤銷上一步
- 查看歷史記錄
- 自訂各名次的分數
- 分數曲線圖
- 快速擷取總分

## 快速開始

### 需要什麼
- Python 3.7 以上
- Windows/macOS/Linux 都可以

### 怎麼安裝

1. **下載專案**
```bash
git clone https://github.com/Aaron001222/KartRider-Score-V0.1d.git
cd KartRider-Score-V0.1d
```

2. **安裝需要的套件**
```bash
pip install -r requirements.txt
```

3. **執行程式**
```bash
python main.py
```

## 怎麼使用

### 基本操作

1. **設定玩家**
   - 輸入玩家名字
   - 選個喜歡的顏色

2. **分配排名**
   - 用下拉選單直接選排名（X=沒完成，1-8=名次）
   - 或者用按鈕：左鍵依序點擊分配名次，右鍵清除

3. **計算分數**
   - 點「計算分數並排序」按鈕
   - 系統會自動算分並更新

### 其他功能

- **撤銷**：點「上一步」可以回到上一個狀態
- **歷史**：點「歷史紀錄」看所有局數的分數
- **曲線圖**：點「分數曲線」看分數變化
- **擷取**：點「擷取」快速顯示總分

### 分數設定

預設分數：
- 第1名：10分
- 第2名：7分
- 第3名：5分
- 第4名：4分
- 第5名：3分
- 第6名：1分
- 第7名：0分
- 第8名：-1分
- 沒完成(X)：-5分

這些分數都可以在「名次分數定義」那邊自己改。

## 按鈕排名小技巧

1. **左鍵點擊**：依序分配1~8名（已經分配的不會重複）
2. **右鍵點擊**：清除該玩家的名次，變回「X」
3. **全視窗右鍵**：清空所有玩家的排名
4. **自動清空**：每局算完會自動清空，準備下一局

## 打包成 exe

想要做成 exe 檔案的話：

```bash
# 安裝 PyInstaller
pip install pyinstaller

# 打包
pyinstaller --noconsole --onefile --icon=NL.ico --add-data "NL.ico;." main.py
```

exe 檔案會在 `dist` 資料夾裡。
